快速开始

Docker 部署（推荐）

bash 
docker run -d \
  --name bbot \
  --restart unless-stopped \
  -p 5000:5000 \
  -p 8888:8888 \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -v /your/data/path:/app/mount \
  241793/b-bot:latest

访问面板

部署完成后，打开浏览器访问 http://localhost:5000

配置 QQ 对接

在 NTQQ 的 LLOneBot 插件中配置 WebSocket：

ws://127.0.0.1:8888/ws/qq

插件开发

最小插件模板

python 
"""
示例插件
这是一个示例插件
"""

__description__ = "示例插件"
__version__ = "1.0.0"
__author__ = "开发者"
__imType__ = "qq"
__admin__ = False

async def handle_message(msg, middleware):
    content = msg.get("content", "")
    if content == "你好":
        return {"content": "你好！我是 B-BOT"}

def register(middleware):
    middleware.register_message_handler(handle_message)
